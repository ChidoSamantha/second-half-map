# Second Half Map — Wheel of Priorities

Interactive assessment tool for the Second Half Map RetirementLife Workshop Series.

## Deployment

This is a single-page static site. To deploy:

1. Upload this entire folder to v0 or Vercel
2. It will deploy automatically as a static site
3. No build step, no dependencies, no configuration needed

## Files

- `index.html` — the complete self-contained tool (all CSS and JS inline)
- `vercel.json` — Vercel deployment configuration

## Future tools

As new tools are built (values card sort, postcard activity, circles of relationship map),
they will be added to this same folder as additional HTML files and linked from a central index.
